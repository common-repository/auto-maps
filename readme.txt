Automatically turn address tags into google maps embed code.

== Description ==
Automatically turn address tags into google maps embed code.

Example: 
`<address data-width="300" data-height="250">
1600 Amphitheatre Pkwy, Mountain View, CA 94043, United States
</address>`

For more information visit plugin page <a href="http://www.idiotinside.com/automaps-wordpress-plugin/">Auto Maps</a>

== Installation ==


1. Upload auto-maps folder to the /wp-content/plugins/ directory
2. Activate the plugin through the 'Plugins' menu in WordPress
